# Hotline Inquiry Tracking Program

This program is designed to automate the tracking and management of inquiries made to a hotline. It helps customer service teams efficiently process and respond to inquiries, ensuring timely follow-up and improving overall customer satisfaction.

## Features

- **Create New Inquiry**: Allows you to log a new inquiry, capturing details such as the caller’s information, inquiry type, and urgency.
- **Track Inquiry Status**: The program tracks the status of each inquiry (e.g., pending, resolved, escalated).
- **Search Inquiries**: Enables searching inquiries by various criteria, such as caller name, inquiry type, or status.
- **Categorize Inquiries**: Sort inquiries into categories based on the issue type (e.g., technical support, billing, general questions).
- **Generate Reports**: Create reports for analysis of call volumes, response times, and resolved issues.

## Usage

1. Launch the program.
2. To log a new inquiry, click the “New Inquiry” button and enter the caller’s information, inquiry details, and urgency level.
3. Track the status of each inquiry by updating its status (e.g., “pending,” “resolved”).
4. Use the search function to filter inquiries based on specific criteria (e.g., caller name, inquiry type, or status).
5. Generate reports to analyze the number of inquiries, response times, and the rate of issue resolution.

## Data Structure

An inquiry consists of the following information:

- **Caller Name**: The name of the person making the inquiry.
- **Phone Number**: The caller's contact number.
- **Inquiry Type**: The category of the inquiry (e.g., technical support, billing, general).
- **Date & Time**: When the inquiry was made.
- **Status**: The current status of the inquiry (e.g., pending, resolved, escalated).
- **Priority**: The urgency level of the inquiry (e.g., low, medium, high).
